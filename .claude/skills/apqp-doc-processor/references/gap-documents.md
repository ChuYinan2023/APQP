# 缺口分析：补充文档类型参考

阶段3提取L2零部件特性时，若现有文档无法支撑某条特性的定义，需标记为缺口并指明所需补充文档。

## 常见补充文档类型

| 文档类型 | 说明 | 主要提供的信息 | 典型文件名模式 |
|---|---|---|---|
| 零件图纸（Part Drawings） | 零部件工程图 | 尺寸、公差、几何特性、表面处理、KPC标注 | `*_DWG_*.pdf`, 3D CAD |
| BOM（物料清单） | Bill of Materials | 零部件清单、层级关系、数量、供应商编号 | `BOM_*.xlsx` |
| 零件规格书（Part Specifications） | 零部件级规格文件 | 零部件级性能/功能要求 | `PS_*.pdf`, `PF.*` |
| 材料规范（Material Specifications） | 材料标准文件 | 材料属性、牌号、标准号、IMDS数据 | `MS.*`, `CS.00265` |
| 供应商规格（Supplier Specifications） | 供应商提供的规格 | 供应商能力、过程约束、交付条件 | 供应商名称+型号 |
| 产品卡（Product Card） | OEM内部参数卡 | ECU接口定义、设备分类、具体电气参数 | `PC_*.pdf` |
| 接口定义文件（Interface Document） | 系统接口规格 | Pinout、信号定义、通信协议 | `IF_*.pdf`, EE Checklist |
| 整车布置图（Package Layout） | 安装位置图 | 安装点位、间隙、线束走向 | `PKG_*.pdf` |
| DV/PV报告（Validation Report） | 验证测试报告 | 实测数据、通过/失败记录 | `DVP&R_*.xlsx` |

## 缺口记录格式

每条缺口应记录：

| 字段 | 说明 | 示例 |
|------|------|------|
| 缺口编号 | GAP-{序号} | GAP-01 |
| 缺失信息 | 具体缺什么 | 滤芯具体材料规格（滤材类型/层数/面积） |
| 影响L2特性 | 受影响的L2 ID | L2-2.01~L2-2.17 |
| 关联L1特性 | 上溯到L1 | F-02, F-03 |
| 所需文档类型 | 从上表选择 | 零件规格书 + 供应商图纸 |
| 优先级 | 高/中/低 | 高 |
| 判定依据 | 为什么是这个优先级 | 直接影响核心过滤功能 |

## 缺口检查最小覆盖要求

**每个子零部件至少检查以下4个维度**，若某维度在现有文档中未定义，必须记录为GAP：

| 维度 | 检查内容 | 示例GAP |
|------|---------|---------|
| 材料规格 | 具体材质/牌号/硬度/S规格号 | 滤芯滤材类型/层数/面积未定义 |
| 尺寸/结构 | 关键尺寸、公差、几何特征、壁厚 | 壳体壁厚/安装点位坐标未定义 |
| 接口详图 | Pinout、配合尺寸、连接方式 | 连接器pinout详图未提供 |
| 供应商数据 | 具体型号、供应商能力、BOM条目 | 排水阀结构型式(螺旋/按压)未定义 |

此外，总成级别还须检查：
- BOM完整性（是否有正式BOM清单）
- 重量分类（CS.00056 Weight Classification具体值）

## 优先级判定标准

| 优先级 | 标准 |
|--------|------|
| **高** | 缺口直接影响核心功能(F)或安全(S)特性，阻碍DVP&R编制 |
| **中** | 缺口影响可靠性(R)或环境(E)特性，但可通过假设暂时绕过 |
| **低** | 缺口仅影响辅助特性(A)或保护帽等非核心零部件 |
