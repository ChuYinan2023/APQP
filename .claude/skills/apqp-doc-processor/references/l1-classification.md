# L1工程特性分类体系

7大分类，全程统一使用，不因零部件类型改变。

| 代码 | 分类名称 | 含义 | 典型内容举例 |
|------|---------|------|------------|
| **F** | 功能 (Function) | 产品的核心功能参数 | 过滤效率、流量、压降、制动力、输出功率 |
| **P** | 性能-电气/传感器 (Performance) | 电气/电子/信号相关性能 | 电压电流、传感器特性、ECU接口、通信协议 |
| **M** | 机械/结构 (Mechanical) | 物理结构和机械性能 | 密封性、强度、扭矩、接口尺寸、装配 |
| **R** | 可靠性/耐久 (Reliability) | 寿命和长期表现 | 热循环、脉动、振动、老化、MTBF |
| **E** | 环境/防护 (Environmental) | 环境适应性和防护 | IP等级、温度范围、防腐、盐雾、化学耐受 |
| **S** | 安全/法规 (Safety/Regulatory) | 安全要求和法规合规 | 法规合规、HAZMAT、IMDS、失效安全 |
| **A** | 外观/感知 (Appearance) | 外观和用户感知 | NVH、BSR、表面质量、标识、人体工程学 |

## 分类判定指引

- 同一参数可能跨分类，选择**最直接**的分类作为主分类
- "过滤效率"→ F（核心功能），不是P
- "加热器功率"→ P（电气性能），不是F
- "爆破压力"→ M（机械强度），不是S
- "防腐盐雾"→ E（环境防护），不是R
- 模糊项标注 `[待确认分类]`，暂定主分类
