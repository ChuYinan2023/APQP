# 零部件分解指引

阶段3中将L1工程特性分解到L2零部件特性时，按零部件类型使用不同分解策略。

---

## 零部件类型分类

### 1. 结构件 (Structural Parts)

**典型零件**：壳体(Housing)、支架(Bracket)、盖板(Cover)、底座(Base)

**必须提取的特性维度**：

| 维度 | 具体特性 |
|------|---------|
| 材料 | 材质(PA66-GF/铝/钢)、牌号、S材料规格号、IMDS |
| 尺寸 | 关键尺寸、公差(按QR-10012)、配合面 |
| 强度 | 爆破压力、脉动耐久、崩溃压力 |
| 防护 | IP等级(壳体贡献)、耐腐蚀、耐化学 |
| 外观 | 表面质量、标识标记区域、无裂纹/变色 |
| 环保 | 可回收性、再磨碎比例 |

### 2. 功能件 (Functional Parts)

**典型零件**：滤芯(Filter Element)、阀体(Valve Body)、碗体(Bowl)

**必须提取的特性维度**：

| 维度 | 具体特性 |
|------|---------|
| 核心性能 | 效率/容量/分离率等主功能参数 |
| 流阻 | 压降(新品/寿命末期/低温) |
| 寿命 | 容尘量、老化后性能保持 |
| 清洁度 | 出厂清洁度(颗粒数/重量) |
| 耐温 | 工作温度范围、热循环耐受 |
| 耐化学 | 介质兼容性(燃油类型、FAME含量) |

### 3. 电子件 (Electronic Parts)

**典型零件**：传感器(Sensor)、加热器(Heater)、ECU、执行器

**必须提取的特性维度**：

| 维度 | 具体特性 |
|------|---------|
| 电气参数 | 电压/电流/功率/阻抗/信号特性 |
| 标称参考值 | **传感器特性曲线类参数，必须提取至少1个标称参考点的具体数值**（如NTC@25°C标称电阻2.057kΩ±公差），不可仅写"见Table X" |
| 接口 | ECU接口(pull-up/pull-down电阻、滤波电容)、Pinout |
| 设备分类 | 按CS.00056: 设备类型、功能分类、安装分类、温度分类、振动分类、ESD分类 |
| EMC | 按CS.00244: 传导/辐射发射、抗扰度 |
| 失效安全 | 故障模式(开路/短路/卡死)、安全响应 |
| 寿命 | 循环次数、工作小时数、过压耐受 |

### 4. 密封件 (Sealing Parts)

**典型零件**：O-Ring、垫圈(Gasket)、唇封(Lip Seal)

**必须提取的特性维度**：

| 维度 | 具体特性 |
|------|---------|
| 材料 | 弹性体类型(FKM/NBR/EPDM/HNBR)、硬度(Shore A)、S规格号 |
| 密封性 | 泄漏量(cm³/min)、测试压力、测试介质 |
| 温度范围 | 低温柔韧性、高温压缩永久变形 |
| 介质兼容 | 燃油(EN590)、FAME、发动机舱流体 |
| 老化 | 静态老化后密封保持、动态老化后密封保持 |
| 尺寸 | 内径/线径/公差(按S 14618) |

### 5. 连接件 (Connection Parts)

**典型零件**：燃油接头(Fuel Fitting)、快速接头(Quick Connect)、电气连接器(Connector)

**必须提取的特性维度**：

| 维度 | 具体特性 |
|------|---------|
| 尺寸标准 | SAE J2044、PF.90298、B12系列、APTIV型号 |
| 逐接口分行 | **多个接口点(进口/出口/回油等)的参数必须逐接口分行记录**，因为不同接口可能规格不同 |
| 力矩/力值 | 抗扭矩(Nm)、侧向载荷、插拔力、配合力 |
| 人机工程 | SAE/USCAR-25合规、手力/卡扣力限值 |
| IP等级 | 密封连接器的IP要求 |
| 耐久 | 振动后完整性、脉动后完整性 |

### 6. 辅助件 (Auxiliary Parts)

**典型零件**：保护帽(Protection Cap)、标签(Label)、包装件

**必须提取的特性维度**：

| 维度 | 具体特性 |
|------|---------|
| 覆盖范围 | 需保护的接口列表 |
| 形状规格 | 按附件/图纸 |
| 移除方式 | 手动/工具 |

## BOM结构树推断方法

当无正式BOM时，从文档内容推断零部件分解：

1. **从测试项目反推**：每个独立测试对象通常是一个子零部件
   - 例：加热器效率测试 → 存在独立的加热器子零部件
2. **从接口描述反推**：每个定义了接口的部分通常是一个子零部件
   - 例：液压接口进口/出口 → 存在燃油接头零部件
3. **从材料要求反推**：不同材料要求通常对应不同子零部件
   - 例：O-Ring规格(S 14618) → 存在独立密封件
4. **从设备分类表反推**：CS.00056分类表中列出的设备通常各自独立
   - 例：Heater / WIF Sensor / NTC Sensor → 三个独立电子件

## L2 ID编码规则

```
L2-{零部件序号}.{特性序号}

零部件序号：按BOM树从上到下编号（1, 2, 3...）
特性序号：每个零部件内从01开始递增

示例：
L2-1.01  → 壳体的第1条特性
L2-2.05  → 滤芯的第5条特性
L2-4.12  → 加热器的第12条特性
```

每条L2记录必须包含：
- L2 ID
- 零部件特性名称
- 目标值/范围
- 单位
- 关联L1 ID（可多个）
- 来源文档/章节
