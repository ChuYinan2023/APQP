# 阶段5：质量屋矩阵 (QFD)

**前置条件**: L1和L2均已完成反向校验、无遗漏

## 分块策略

当 L1×L2 矩阵超过 30×30 时，**必须**按功能域分块填写。

分块原则（通用，不绑定具体零部件）：

| 块 | L1范围 | L2范围 | 说明 |
|----|--------|--------|------|
| 块1 | F-xx 功能类 | 与核心功能直接相关的零部件 | 功能域 |
| 块2 | P-xx 性能/电气类 | 电子件、传感器、执行器 | 电气传感域 |
| 块3 | M-xx 机械/结构类 | 结构件、密封件、连接件 | 结构机械域 |
| 块4 | R-xx 可靠性类 | 全部零部件 | 耐久域（横跨） |
| 块5 | E-xx 环境类 | 暴露于环境的零部件 | 环境防护域 |
| 块6 | S-xx 安全类 | 全部零部件 | 安全法规域 |
| 块7 | A-xx 外观类 | 外露零部件 | 外观感知域 |

具体块内L2范围根据实际BOM树和特性清单确定。

## 关联强度判定标准

| 等级 | 符号 | 分值 | 判定标准 |
|------|------|------|---------|
| 强 | ◎ | 9 | L2是L1的**直接技术实现**，L2变化直接决定L1达标 |
| 中 | ○ | 3 | L2对L1有**间接贡献**，是达标的辅助条件之一 |
| 弱 | △ | 1 | L2对L1有**微弱影响**，仅极端条件下关联 |
| 无 | (空) | 0 | 无关联 |

## 执行步骤

1. 确定分块方案（参照上表，结合实际L1/L2清单）
2. 逐块填写 L1×L2 关联矩阵
3. 填写屋顶矩阵：L2之间 ＋正相关(协同) / －负相关(冲突)
4. 补充目标值、重要度权重
5. **全局审查**：
   - 全空行（L1无任何L2关联 → 可能遗漏L2）
   - 全空列（L2无任何L1关联 → 可能多余或遗漏L1）
   - 关联密度最高的零部件（设计关键件）
   - 关键冲突（负相关对，需设计平衡）
