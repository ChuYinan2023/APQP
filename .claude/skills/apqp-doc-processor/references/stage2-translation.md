# 阶段2：翻译解读 → 核心内容结构树

## 步骤

1. 分页/分段逐批翻译解读（每批5-10页）
2. 每批输出阶段性摘要，累积拼接
3. 边翻译边提取关键信息，构建结构大纲
4. 全部完成后全局结构回顾
5. 建立术语对照表（中英），不确定翻译标注 `[待确认]`

## 结构树格式

```
1. 文档名称
   1.1 章节名称
       - 核心内容摘要
       - 关键参数：xxx
   1.2 ...
```

## 术语表格式

| 英文 | 中文 | 备注/上下文 |
|------|------|------------|
| Water-in-Fuel (WIF) | 油中水传感器 | 检测燃油中水含量 |
