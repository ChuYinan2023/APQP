# 零部件知识：柴油滤清器 (Diesel Fuel Filter)

<!-- 更新于 2026-02-16 -->

## 基本信息

- **零部件大类**：燃油系统 - 过滤模块
- **典型应用**：柴油发动机燃油系统，位于油箱和高压泵之间
- **核心功能**：过滤燃油中的颗粒和水分，保护高压共轨系统

## 典型BOM结构

```
柴油滤清器总成 (Diesel Filter Module)
├── 1. 壳体 Housing (结构件)
├── 2. 滤芯 Filter Element (功能件) ← 核心件
├── 3. 储水碗 Water Collection Bowl (功能件)
├── 4. 加热器 Heater (电子件)
├── 5. WIF传感器 Water-in-Fuel Sensor (电子件)
├── 6. NTC温度传感器 (电子件)
├── 7. 密封件 O-Rings/Gaskets (密封件, 多个)
├── 8. 燃油接头 Fuel Fittings (连接件, 进/出口)
├── 9. 电气连接器 Electrical Connector (连接件)
├── 10. 排水阀 Drain Valve (功能件)
└── 11. 保护帽 Protection Caps (辅助件)
```

## 核心工程参数

| 参数 | 典型范围 | 关键标准 |
|------|---------|---------|
| 过滤效率 | ≥93%@4μm (EN12259标准) | ISO 4020, ISO 19438 |
| 压降（新品） | ≤15-25 kPa @额定流量 | 与流量/温度组合相关 |
| 容尘量 | ≥30-50g (按ISO 4020) | 决定换滤芯周期 |
| 水分离效率 | ≥95% | ISO 4020 Annex A |
| 爆破压力 | ≥6-10 bar | 安全裕度通常3x工作压力 |
| 工作温度 | -40°C ~ +120°C | 低温启动是关键场景 |
| 加热器功率 | 200-500W | 低温时融蜡/加热燃油 |
| IP等级 | IP6K9K (发动机舱) | 壳体+连接器联合贡献 |

## 常见缺口

1. **滤芯详细规格** — 通常需要供应商图纸/零件规格书才能完整定义滤材参数
2. **零件图纸** — 壳体/接头尺寸公差需要工程图
3. **BOM清单** — 正式BOM通常不在SPEC中，需要单独获取
4. **产品卡** — ECU接口具体参数（pull-up/pull-down电阻值等）
5. **材料规范** — O-Ring/密封件的具体材料牌号和S规格号

## 工程权衡

1. **过滤效率 vs 压降**：效率越高，压降越大，需要平衡
2. **容尘量 vs 体积**：大容尘量需要更大滤芯面积
3. **低温性能 vs 成本**：加热器功率增大提升低温性能但增加成本和电力消耗
4. **密封性 vs 可维护性**：更强的密封可能使滤芯更换更困难
5. **IP防护 vs 散热**：更高IP等级可能影响散热

## 处理注意事项

1. 过滤效率通常按多个粒径(4μm, 6μm, 12μm等)分别给出，应分行记录
2. 压降参数通常有多个工况组合（流量×温度×新品/寿命末期），每个组合单独记录
3. 低温相关特性是柴油滤的特殊关注点（石蜡析出问题）
4. 水分离功能是柴油滤区别于汽油滤的关键特性
5. FAME(脂肪酸甲酯/生物柴油)含量对材料兼容性有重要影响
