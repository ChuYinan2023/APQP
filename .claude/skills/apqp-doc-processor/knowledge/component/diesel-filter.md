# 零部件知识：柴油滤清器 (Diesel Fuel Filter)

<!-- 更新于 2026-02-24，FILTER项目验证后补充 -->

## 核心功能

加热燃油滤清水分离器 (Heated Fuel Filter Water Separator, HFFWS)。过滤燃油中的颗粒和水分，保护高压共轨喷射系统。位于油箱和高压泵之间。含电加热功能应对低温石蜡析出。

## 典型BOM结构

```
柴油滤清器总成 (Smart Fuel Filter)
├── 壳体 Housing ← 承压主体，耐腐蚀/耐脉动
├── 滤芯 Filter Element ← 核心件，决定过滤效率/压降/容尘量
├── 集水碗 Water Collection Bowl ← 水分离+储水+排水阀
├── PTC加热器 Heater ← 低温融蜡/加热燃油，含恒温器
├── WiF传感器 Water-in-Fuel Sensor ← 水位检测，主动/被动式
├── NTC温度传感器 ← 温度测量，SENT协议通信
├── 密封件 Seals/Gaskets ← 外部O-Ring + 滤芯垫片
├── 液压快接 Quick Connect Fittings ← 进/出口，SAE J2044
├── 电气连接器 Electrical Connector ← 通常6-pin（+5V/WiF/PTC+/PTC-/SENT/GND）
└── 保护帽 Protection Caps ← 运输/存储时的接口保护
```

注：排水阀通常集成在集水碗内，不单独作为BOM子项。

## 常见缺口（按优先级）

### 高优先级
1. **工程图纸** — 尺寸公差、关键尺寸、配合关系（几乎必缺）
2. **E/E环境规范** (CS.00056/CS.00263) — 环境试验详细profile、设备分类
3. **EMC规范** (CS.00244) — EMC/EMI测试项目和限值

### 中优先级
4. **Product Card** — NTC特性曲线和WiF传感器的正式值（通用Spec只给"example"）
5. **快接规范** (PF.90298) — 液压接口详细密封/配合要求
6. **腐蚀规范** (CS.00251) — 详细cycle参数
7. **CTS附件** — 3D模型构型、安装方式
8. **传感器型号/规格** — 供应商数据

### 低优先级
9. **滤芯详细规格** — 滤材类型/层数/过滤面积（供应商内部数据）
10. **保护帽规格文件** — 通常有单独xlsx定义

## 处理注意事项

1. **过滤效率必须多粒径分行**：4μm/5μm/10μm/15μm/20μm/30μm各自一行，不要合并
2. **压降有多个工况**：标称流量(CTS) vs 通用流量(通用Spec) vs 低温工况，每个单独记录
3. **低温是核心关注点**：石蜡析出(CFPP)、加热器效率、低温密封——这些是柴油滤区别于其他滤清器的关键
4. **水分离功能**：自由水(ISO 16332) vs 乳化水(ASTM D971)是两个不同测试，分别记录
5. **FAME(生物柴油)**：EN590+30%FAME是材料老化的标准条件，影响密封件和滤芯
6. **传感器参数**：通用Spec中的NTC曲线/WiF参数常标"example"，实际值在Product Card
7. **3个独立电子设备**：加热器、WiF传感器、NTC传感器在设备分类表(Table 4)中各自独立，防护等级/ESD/振动等级需分别记录
8. **Pinout是C类特性**：6-pin连接器的每个PIN定义都是关键特性
9. **QFD关键件**：滤材规格和壳体是关联密度最高的零部件（设计核心）

## OEM特有要求

### Stellantis
- 文档组合：PF.90150(通用) + CTS(项目具体值)，CTS优先
- 过滤效率：多通(ISO 19438)为主，CTS可能额外指定初始效率(如98%@4μm)
- 电子设备按CS.00056/CS.00263做设备分类（加热器、WIF、NTC各自独立分类）
- EMC按CS.00244，需S EE/E部门同意验证方案
- DV/PV样本量在Annex A (Table A-1)：功能15+15(P99C90)，耐久45+45(R95C90)
- CC频次在Annex A-2 (Table A-2)：密封100%、加热器/WiF周检、效率/容尘/爆破季检
- 恒温器关断和失效安全是强制测试项
- 保护帽取消需S品质部(Plant UTE)书面同意
