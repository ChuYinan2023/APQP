# 通用零部件知识（首次处理新类型时使用）

当 `knowledge/component/{type}.md` 不存在时，以此文件作为基线。

---

## 阶段0 首次处理提示

标注：**首次处理该零部件类型，知识库暂无积累，按通用流程执行，完成后回写经验。**

---

## 通用BOM推断原则

无正式BOM时，按以下4条从文档内容推断：

1. **从测试项目反推** — 每个独立测试对象 → 一个子零部件
2. **从接口描述反推** — 每个定义了接口的部分 → 一个子零部件
3. **从材料要求反推** — 不同材料要求 → 不同子零部件
4. **从设备分类表反推** — 分类表中列出的设备 → 各自独立

## 通用L1特性分布基线

| 分类 | 通用预估数量 | 必须覆盖的方向 |
|------|------------|--------------|
| F | 10-25 | 核心功能参数（效率/流量/输出/容量） |
| P | 5-20 | 电气/信号参数（如有电子件） |
| M | 8-15 | 密封/强度/关键尺寸/装配 |
| R | 8-15 | 热循环/振动/脉动/寿命 |
| E | 8-12 | IP等级/温度范围/防腐/介质兼容 |
| S | 3-6 | 法规合规/IMDS/失效安全 |
| A | 3-8 | 外观/NVH/标识 |

---

## 通用子零部件类型 × 必查维度

| 子零部件类型 | 必须覆盖的特性维度 |
|------------|----------------|
| 结构件（壳体/支架/盖板） | 材料/强度/尺寸公差/防护/外观 |
| 功能件（滤芯/阀/碗体） | 核心性能/流阻/寿命/清洁度/耐温/耐化学 |
| 电子件（传感器/加热器/ECU） | 电气参数/标称参考值/接口/设备分类/EMC/失效安全/寿命 |
| 密封件（O-Ring/垫圈/唇封） | 材料/密封性/温度范围/介质兼容/老化/尺寸 |
| 连接件（接头/连接器） | 尺寸标准/逐接口分行/力值/耐久 |
| 辅助件（保护帽/标签） | 覆盖范围/规格/移除方式 |

## 通用缺口四维度检查

每个子零部件至少检查：

- [ ] 材料规格（具体材质/牌号/硬度/S规格号）
- [ ] 尺寸/结构（关键尺寸、公差、几何特征）
- [ ] 接口详图（Pinout、配合尺寸、连接方式）
- [ ] 供应商数据（具体型号、供应商能力）

总成级额外检查：

- [ ] BOM完整性
- [ ] 重量分类
