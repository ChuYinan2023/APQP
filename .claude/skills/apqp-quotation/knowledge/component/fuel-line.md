# 零部件知识：燃油管 (Fuel Line / Fuel Bundle)

<!-- 更新于 2026-03-02, KP1 FBFS 项目回写 -->

## 核心功能

从燃油滤清器/油箱到发动机/喷油器的燃油输送管路总成，需满足高压密封、耐燃油腐蚀、静电消散、蒸发排放控制等要求。

## 典型BOM结构

```
燃油管总成 (Fuel Line Assembly)
├── 尼龙管 (Nylon Tubing) — PA12 多层/单层，主体管路
│   └── 多层结构：外层(PA12) + 阻隔层(EVOH/PA6) + 内层(PA12)
├── 金属管 (Steel Tubing) — 低碳钢，弯管段
├── 快速接头 QC (Quick Connector) — 塑料/金属壳体
│   ├── O-ring (FKM) — 密封
│   ├── 保持卡扣 (Retainer Clip) — PA66 GF30
│   └── 二次锁扣 (Secondary Latch) — 新设计强制
├── 阻尼器 (Damper) — 消除压力脉动
├── 热塑性弹性体保护套 (TPE Cover) — 防磨损
├── 线束卡扣 (Wiring Clip) — 固定
└── 弹簧卡扣 (Spring Clip) — 紧固
```

## 常见缺口

1. **CD.80064 核心文件包** — Stellantis 燃油管路的基础文件包，几乎每次都缺，需向客户索取
2. **SD-11597 燃油兼容清单** — 决定需测试哪些燃油类型（C5/C6/C7等），缺此无法确认测试矩阵
3. **SD-M0008/03 尼龙管审批** — 多层尼龙管材料必须在 Stellantis 批准名单内
4. **CS.00251 腐蚀要求** — J2334 腐蚀测试的具体参数和周期
5. **QR.00001 GPAT** — 验证样件数量可能引用 PF.90068 Annex A 的方法论
6. **车型特定振动频谱** — 通用频谱可用于初始 FEA 和 Z 轴测试，但最终 DV 需车型特定频谱
7. **ESD/MiniSHED 测试能力** — 多数 Tier1 供应商不具备，需外包

## 处理注意事项

1. **多条件必须分行** — 拉脱力在不同温度/管径条件下有不同要求值，插入力按管径分档，爆破压力按温度分档，绝不合并
2. **脉动等级区分** — P1-P5 按工作压力和应用场景分，柴油回油管通常 P1/P5，GDI 通常 P3/P4
3. **区域差异** — CaCl₂ 仅 EMEA，VLD 100% 仅燃油管（蒸气管仅北美），汽油温度 60°C 仅北美
4. **QC vs 管路** — 快速接头 QC 的性能指标来自 PF.90298，管路整体指标来自 PF.90197，两者独立验证
5. **老化后指标** — 拉伸强度/伸长率有"新件"和"老化后"两组指标，老化后为"降幅≤X%"格式
6. **保护套** — Stellantis 有额外保护套要求（拆卸力、颜色对比、密封），常嵌入在 CTS 的 OLE 对象中
7. **阻尼器** — 若供应商同时供应阻尼器，DV 测试需覆盖阻尼器性能
8. **阻尼器 BOM** — 典型阻尼器子零件：端盖(PA66+PA6) + 压缩弹簧(EN 10270-3) + 堵头(PA66 GF30) + 膜片(特种弹性体) + 壳体(PA66 GF30)
9. **QC 爆破章节** — PF.90298 中 QC 爆破压力引用章节为 §7.6 (Elevated Temperature Burst)，非 §6.3.6
10. **Cartridge 插入力** — 筒式 QC (cartridge) 最大插入力 ≤650N（O-ring 120°C/24hr 干燥后轻质矿物油润滑），与普通 QC 插入力标准不同
11. **QC rocker 验收** — QC rocker 测试验收标准为 R90C90（低于一般耐久的 R95C90），45DV/15PV

## OEM特有要求

### Stellantis (FCA)
- 文档组合模式：SSTS(xlsx, 内嵌CTS+TDR) + PF标准(pdf) + 零件描述(pptx) + DVPR模板(xlsx)
- 特有要求：NX+TeamCenter CAD 能力、IMDS 材料认证、Fiat 07416 标记、二次锁扣(新设计强制)
- 典型缺口：CD.80064、SD-11597、SD-M0008/03 几乎必缺
- 客户模板：TDR 文档中嵌入 5 个标准模板（ED&D成本/团队/要求/RASI/BOM），必须递归提取才能发现
- 验收标准：大多数 P99C90，耐久类 R95C90，安全项 All Must Pass
- TESIS 可靠性：按安装位置分区（底盘/机舱），3/12/24/36个月阶梯式目标
